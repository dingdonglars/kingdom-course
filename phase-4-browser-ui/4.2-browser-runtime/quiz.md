# Quiz — Module 4.2

## 1. What's the DOM?

a. A C# library
b. The Document Object Model — the live in-memory tree of your page; manipulable from JS
c. A type of database
d. A font

## 2. What does `await fetch(url)` do?

a. Fires the request and immediately returns
b. Pauses execution until the HTTP response arrives, then returns a `Response` object
c. Throws an error
d. Returns a string

## 3. Why do you need `async` on a function that uses `await`?

a. Speed
b. `await` can only appear inside `async` functions; the marker tells the compiler to handle the suspending/resuming
c. Required by Chrome
d. Optional

## 4. What's CORS?

a. A bug
b. Cross-Origin Resource Sharing — browser security that requires the server to declare which origins may call it; otherwise the browser blocks the response from JS
c. A logging library
d. An auth scheme

## 5. The lesson says "open DevTools every time you work on a page." Why?

a. Tradition
b. Elements (DOM tree), Console (live JS), Network (every request) — three views into what's actually happening. Hiding them means flying blind.
c. Required by browsers
d. To impress reviewers
