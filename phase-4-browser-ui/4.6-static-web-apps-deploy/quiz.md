## 2. Why can't you use `AllowAnyOrigin()` with `AllowCredentials()`?

a. They're incompatible. Browser security: if credentials (cookies) are sent, the server must explicitly name the allowed origins — wildcard is too risky.
b. Performance
c. Required by Azure
d. They work fine together

## 4. What's a CDN?

b. Content Delivery Network — your files cached at edge servers worldwide; users get them from the nearest one
c. A logging library
