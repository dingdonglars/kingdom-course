# Module 5.6 starter — visual world

Studio:
- Insert a Script in `ServerScriptService` named `World`; paste `scripts/server/world.lua`.
- The world script generates a 5x5 grid of tiles on play; click any to build a farm.

For prettier visuals, drag a free model from View → Toolbox into `ServerStorage` and `:Clone()` it instead of `Instance.new("Part")`.
