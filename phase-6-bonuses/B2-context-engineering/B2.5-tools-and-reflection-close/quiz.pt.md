## 2. A lição recomenda "um no estilo chat mais um inline" para um projeto de aprendizado. Por que não as cinco ferramentas de uma vez?

- **a.** Cada ferramenta tem custo de configuração, modelo mental, gerenciamento de chaves e cobrança — duas ferramentas que você domina batem cinco que você mal lembra
- **b.** Limites de licença impedem rodar mais de duas ferramentas de IA ao mesmo tempo
- **c.** Desempenho — rodar várias ferramentas de IA deixa o notebook mais lento
- **d.** Exigido pelos termos de serviço da Anthropic

## 3. O que é MCP?

- **a.** Microsoft Compiler Pipeline — um sistema de build para trabalho em C# multiplataforma
- **b.** Model Context Protocol — o padrão emergente da Anthropic para ferramentas com que as IAs podem conversar (bancos de dados, sistemas de arquivos, APIs web)
- **c.** Multi-Component Pattern, uma abordagem de design de software
- **d.** Uma versão específica do .NET lançada junto com o .NET 10

## 4. A lição diz "a disciplina é sua; as ferramentas vêm e vão." O que isso quer dizer?

- **b.** A IA é pouco confiável e não se deve depender dela
- **c.** Tradição — desenvolvedores mais velhos preferem fazer tudo na mão
- **d.** Estilo obrigatório para a mensagem de encerramento do currículo
