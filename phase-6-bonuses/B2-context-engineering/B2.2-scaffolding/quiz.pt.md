## 1. O que é um arquivo de scaffold?

- **a.** Um documento de nível de projeto que a IA lê no início da sessão (ex.: `ARCHITECTURE.md`, `STANDARDS.md`, `CLAUDE.md`)
- **b.** Um novo tipo de projeto C# usado para trabalho relacionado a IA
- **c.** Um arquivo de fixture de teste exigido pelo teste de integração de IA
- **d.** Qualquer arquivo que tenha a palavra "scaffold" no nome

## 2. Por que ser *curto* é uma virtude para arquivos de scaffold?

- **a.** Cada linha é lida toda sessão — enchimento desperdiça tokens e atrasa as partes de que a IA de fato precisa
- **b.** Arquivos rodam mais rápido quando são curtos, até no disco
- **c.** Exigido pelo Claude; arquivos mais longos são rejeitados
- **d.** Preferência de estilo pessoal, não uma restrição real

## 3. O que um arquivo de exemplo em `.claude/examples/` te dá?

- **a.** Um trecho escolhido a dedo, no estilo certo, para o qual você pode apontar a IA — *"siga o estilo deste exemplo"*
- **b.** Um conjunto de casos de teste que a IA pode rodar contra o seu código
- **c.** Documentação só para leitores humanos; a IA não lê esses arquivos
- **d.** Nada de útil; a IA ignora arquivos fora dos locais padrão dela

## 4. Por que "scaffolds desatualizados são piores que nenhum scaffold" é um aviso de verdade?

- **a.** Um scaffold errado engana a IA com confiança — ela o segue até a resposta errada em vez de perguntar
- **b.** Scaffolds desatualizados parecem pouco profissionais numa revisão de código
- **c.** Exigido pelo runtime do .NET que ele esteja atualizado
- **d.** Scaffolds desatualizados rodam mais devagar que os atuais

## 5. O que o cabeçalho de arquivo "você está aqui" faz?

- **a.** Um comentário de cinco linhas no topo do arquivo dizendo o papel do arquivo, as convenções e os arquivos relacionados — orientação para qualquer leitor que abra ele no meio de uma tarefa
- **b.** Um cabeçalho obrigatório exigido pelo Visual Studio para arquivos de classe
- **c.** Uma tática de otimização para mecanismos de busca em repositórios de código
- **d.** Nada de especial; é só um comentário
