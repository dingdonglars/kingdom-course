# Quiz — Módulo 3.0

> Não escreva suas respostas neste arquivo — abra o `journal/quiz-notes.md` e escreva lá.
>
> Tente primeiro o `quiz.md` em inglês. Use este aqui só quando uma palavra te travar.

## 1. Por que ler o código de outras pessoas antes de escrever o seu?

- **a.** Para você poder copiar ele para o seu projeto
- **b.** Ler constrói discernimento — saber por que uma estrutura é melhor que outra. Tutoriais ensinam sintaxe; ler ensina bom gosto.
- **c.** Porque o currículo exige
- **d.** Para passar o tempo antes de o trabalho de verdade começar

## 2. Mais ou menos qual é a proporção certa de leitura para escrita para um desenvolvedor experiente?

- **a.** Cerca de cinco vezes mais leitura que escrita — ele percorre o código, acha o arquivo certo, percebe as peças que faltam
- **b.** Mais ou menos igual — um para um
- **c.** Quase só escrita; ler é para iniciantes
- **d.** Quase só leitura; escrever é para iniciantes

## 3. O protocolo das cinco perguntas pede que você ache *uma coisa que te surpreenda*. Por quê?

- **a.** Para deixar a anotação do diário mais longa
- **b.** A surpresa é o vão entre o que você supôs e o que está de fato lá — exatamente o ponto onde um novo entendimento se fixa
- **c.** É um hábito emprestado de resenhas de livros
- **d.** Porque o xUnit exige

## 4. Por que a lição diz para *não procurar respostas enquanto você lê*?

- **a.** Para economizar banda em conexões lentas
- **b.** Tentar primeiro, e depois conferir, constrói discernimento. Procurar na hora importa a resposta sem o esforço mental.
- **c.** Porque as respostas na internet estão erradas
- **d.** Para não infringir direitos autorais

## 5. Ler, anotar, e *commitar* as anotações — por que a parte do commit é importante?

- **a.** Ler-e-anotar é trabalho de verdade; deixar isso visível no seu log do git trata como qualquer outra tarefa e deixa um rastro
- **b.** O GitHub vai recusar commits sem uma entrada de diário
- **c.** É uma decoração — fique à vontade para pular
- **d.** Para se preparar para o desafio do M4
