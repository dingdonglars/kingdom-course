# Block 1 — Spark Week

> **Ship four playable toys in a month.**

**What you'll have at the end:** A repo with four small console apps you wrote — a snarky roast generator, a sarcastic guessing game, a 3-room text adventure, and a polished version of one of them with ASCII art and saved files. Plus a README you wrote yourself describing each one.

**Estimated effort:** 4 weeks at 4–6 hrs/week  ·  **Wraps:** M0 *The Joke Toolbox*

---

## Why this block?

Most programming courses start with theory. *"Here's what a variable is. Here's what a loop is."* By week three you've forgotten why you signed up. We do the opposite: you make stuff in the first hour, and we name the concepts after you've already used them. **Magic first. Theory after.**

The four toys are deliberately silly. Sillness sticks. By the end of the month you've shipped four real things that run on your computer, and you've used variables, loops, conditionals, lists, methods, and files — without anyone explaining what they "are" first. Foundations comes next.

## What you'll learn (named)

- That code can do things — it's not magic, you wrote it
- Variables, strings, methods (named after you've used them)
- Loops, conditionals, randomness
- Lists and basic organisation
- Files (preview)

## What you'll build (the climb)

- **Week 1:** install your tools, create your first repo, write a roast generator that prints customisable insults to your friends.
- **Week 2:** a sarcastic Number Guess game that says rude things when you guess wrong.
- **Week 3:** a tiny 3-room text adventure (your own setting).
- **Week 4:** polish — ASCII art, color, save your roast list to a file.

## Brag-worthy outcome

A repo with four tiny games + a README you wrote yourself. The kind of thing you can show a friend and say *"I made this."*
