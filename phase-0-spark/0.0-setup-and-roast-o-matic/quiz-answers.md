`dotnet run` is the command-line tool that takes your C# source code, builds it (turns it into something the computer can execute), and runs the resulting program. The other answers are jokes — though *(d)* would be a useful command to add later.

## 2. c

`git push` *uploads* a copy of your work from your computer to GitHub. The original stays on your computer. So now there are two copies: one local, one on GitHub. **That's the point of git.**
