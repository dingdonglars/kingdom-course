O quiz de hoje é mais uma comemoração. Escolha a resposta certa, mas a verificação de verdade é: você escreveu código hoje, você fez o push dele para o GitHub, e ele está mesmo lá quando você olha? Se sim — você passou.

## 1. O que `dotnet run` faz?

- **a.** Envia seu código para a Microsoft revisar
- **b.** Compila seu código C# e executa o resultado

- **d.** Liga para o Lars no telefone

## 2. Depois de `git push`, onde está o seu código?

- **a.** Só no seu computador
- **b.** Só no GitHub
- **c.** No seu computador e no GitHub, nos dois
- **d.** Na nuvem, com certeza, talvez, quem sabe

## 3. Você acabou de escrever um programa. Você é um programador agora?

- **a.** Ainda não — você precisa de um diploma
- **b.** Ainda não — você precisa saber mais linguagens
- **c.** Sim — programadores são pessoas que escrevem programas
- **d.** Só nos fins de semana
