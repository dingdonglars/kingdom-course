# Module 1.4 starter — game loop

This starter contains **only the files that change** from Module 1.3. To use it:

1. Copy your 1.3 working folder to a new "1.4" folder.
2. Replace the four files below with the versions in this `starter/`:
   - `Kingdom.Engine/Building.cs`
   - `Kingdom.Engine/Kingdom.cs`
   - `Kingdom.Console/Program.cs`
   - `tests/Kingdom.Engine.Tests/KingdomTickTests.cs` (NEW — add this file)
3. `dotnet build` (expect 0 errors)
4. `dotnet run --project Kingdom.Console` (expect Day 1 → Day 6 output)
5. `dotnet test` (expect 16 passing — 11 from 1.3 + 5 new)

If anything fails, read the lesson before debugging.
