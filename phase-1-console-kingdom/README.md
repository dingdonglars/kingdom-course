# Phase 1 — Console Kingdom

> **Your own tycoon game runs in the terminal.**

**What you'll have at the end:** A playable console version of the Kingdom — buildings, resources, a ticking game loop, citizens, random events. You name the kingdom, play a 50-turn run, win or lose. With a real engine code library and your first unit tests on top of it.

**Estimated effort:** 9 weeks at 4–6 hrs/week  ·  **Wraps:** **M2** *Kingdom v1 — Console*

---

## Why this phase?

This is where the *Kingdom* — the project you'll keep coming back to all year — actually starts. Every phase from here is the same Kingdom in a different runtime: console, then saved to disk, then on the web, then in a browser, then on Roblox. The engine you write in this phase is the engine you'll still be using in nine months.

Phase 1 also introduces two of the most important habits in real software. The first is **engine vs shell** — the kingdom's rules live separately from anything to do with the screen. The second is **testing** — you write code that proves your code works. Neither shows up as theory. They show up the moment you'd have written them yourself if you'd thought of them.

## What you'll learn (named)

- Object-oriented programming: classes, properties, constructors, encapsulation
- The engine vs shell discipline (your first deliberate refactor)
- Unit testing with xUnit + Shouldly
- Inheritance and interfaces
- LINQ for asking the engine questions in one line
- Polymorphism via random events
- FakeItEasy for testing code that uses randomness or time
- Code organisation across multiple files and projects

## What you'll build (the climb)

- **Weeks 7–8:** the basics of classes, then split the engine out from the console.
- **Week 9:** unit tests arrive; the game loop starts ticking.
- **Weeks 10–13:** building types, queries, random events, deterministic test fakes.
- **Weeks 14–15:** code organisation, then polish the whole thing into a playable Kingdom v1.

## Brag-worthy outcome

A playable game *you wrote*, that you can demo to anyone in 30 seconds. *"Yeah, I made a tycoon. Build farms, mines, deal with bandits, win or lose."*
