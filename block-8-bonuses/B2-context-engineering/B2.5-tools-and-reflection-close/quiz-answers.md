# Quiz answers — B2.5

## 1. a
Claude Code and Cursor's Agent mode both write/edit/run across multiple files. Inline Copilot completes one line at a time; chat-style talks but doesn't act on the filesystem alone. Pick the right shape for the task.

## 2. a
Each tool has setup cost, mental model, key management, billing. Two tools you know cold beat five you barely remember. Specialise. The shape of "chat for thinking + inline for typing" covers ~all daily needs.

## 3. b
Model Context Protocol (MCP) is Anthropic's open standard for tools-talking-to-AIs. Lets a chat AI use a calendar, a search API, your filesystem — same plug shape, multiple sources. Emerging; worth learning if you build AI tooling.

## 4. a
The model providers will iterate. The model itself will get better. **Your discipline transfers.** Scaffolding, scoping, eval — the skill is durable; the tool is replaceable. Optimise for the durable thing.

## 5. a
"You're a programmer. Past tense, future tense, both — the artefact is the proof." The curriculum's last sentence. Read M5.8's lesson again to feel the close.
