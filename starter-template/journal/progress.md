# Progress

> One line per lesson — date, what you built, one thing you learnt. Same idea as `wins.md` and `quiz-notes.md`. Write it down so it's real.

## How to use this file

After each lesson, add a line at the bottom in this format. Worked example from a future you:

```
Module 1.4 — Game Loop — 2026-MM-DD — added Tick to Building and AdvanceDay to Kingdom, ran 5 days in a console loop. Learnt: a "side effect" is a method that changes state instead of returning a value — most game logic is side effects.
```

The "Learnt" sentence is the important part. One thing that stuck, surprised you, or felt clearer than yesterday. If nothing stuck, write that and bring it to the weekly sync.
